## ⚠️ **문제 해결 (빠른 참조)**

### 사이트가 안 열려요
```
1. 브라우저 캐시 초기화 (Ctrl+Shift+Delete)
2. 시크릿 창에서 시도
3. 2~3분 후 재시도
```

### 새 기능이 안 보여요
```
1. 강력 새로고침 (Ctrl+Shift+R 또는 Cmd+Shift+R)
2. 다른 브라우저에서 시도
3. GitHub Actions 배포 완료 확인
```

### 배포 스크립트 오류
```
1. Git 설치 확인: git --version
2. 저장소 폴더 확인: ls -la .git
3. 인터넷 연결 확인
```
