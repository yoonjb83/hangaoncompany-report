## 🤖 배포 자동화 스크립트

### deploy.ps1 (Windows)
**플랫폼:** Windows PowerShell  
**기능:**
- 자동 Git 스테이징
- 자동 커밋 생성
- 자동 GitHub 푸시
- 배포 상태 안내
- 브라우저 자동 열기

**사용법:**
```powershell
cd /path/to/hangaon-report
.\deploy.ps1
```

---

### deploy.sh (Mac/Linux)
**플랫폼:** Mac, Linux Bash  
**기능:**
- 자동 Git 스테이징
- 자동 커밋 생성
- 자동 GitHub 푸시
- 배포 상태 안내
- 브라우저 자동 열기 (macOS)

**사용법:**
```bash
cd /path/to/hangaon-report
./deploy.sh
```

---

### verify.sh (Mac/Linux)
**플랫폼:** Mac, Linux Bash  
**기능:**
- 20개 항목 자동 검증
- 인터랙티브 체크리스트
- 통과율 계산
- 결과 요약

**사용법:**
```bash
./verify.sh
```
