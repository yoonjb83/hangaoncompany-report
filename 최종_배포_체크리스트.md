## 📋 트러블슈팅

### 문제: 배포가 실패함
**증상:** GitHub Actions에서 빨간 X 표시  
**해결:**
1. Actions 탭의 실패한 작업 클릭
2. 로그 상세 내용 확인
3. 에러 메시지 근거로 수정

### 문제: 새 기능이 안 보임
**증상:** 기존 사이트와 동일하게 보임  
**해결:**
- 브라우저 캐시 초기화: Ctrl+Shift+Delete
- 시크릿 창에서 테스트
- 3~5분 후 다시 새로고침

### 문제: 리포트 저장이 안 됨
**증상:** 버튼을 클릭해도 아무 일도 안 함  
**해결:**
- F12 키 → Console 탭에서 에러 확인
- 다른 브라우저에서 시도
- 페이지 새로고침 후 재시도

### 문제: 포커스 전환이 작동 안 함
**증상:** Enter 키를 눌러도 다음 필드로 이동 안 함  
**해결:**
- JavaScript가 활성화되어 있는지 확인
- 브라우저 개발자 도구(F12)의 Console에서 에러 확인
- 다른 브라우저에서 테스트

### 문제: 도메인 변경 후 기존 URL이 작동 안 함
**증상:** hangaonco-kmex URL에 접속 안 됨  
**해결:**
- GitHub의 자동 리다이렉트 대기 (최대 24시간)
- 모든 링크를 새 도메인으로 업데이트

> 더 많은 해결 방법: `수정사항_적용_가이드.md`의 "문제 해결" 섹션 참고
